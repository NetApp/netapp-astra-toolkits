Create a backup, protection policy, or snapshot on one or more running compute clusters managed by NetApp Astra Control.

Usage: toolkit.py create [ARGUMENT] [OPTION]

Arguments:
  backup            Create a backup
  protectionpolicy  Create a protection policy
  snapshot          Create a snapshot

Options:
  -h, --help            Show this help message and exit.

EXAMPLES

Getting Help

Read the help message for creating a backup:

  ./toolkit.py create backup -h

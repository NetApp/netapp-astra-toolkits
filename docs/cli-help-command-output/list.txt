List the apps, backups, or clusters being managed by NetApp Astra Control.

Usage: toolkit.py list [ARGUMENT] [OPTION]

Arguments:
  apps            List apps
  backups         List backups
  clusters        List clusters

Options:
  -h, --help            Show this help message and exit.

EXAMPLE

List the apps being managed by your NetApp Astra Control account:

  ./toolkit.py list apps

Read the help message for listing the apps being managed by your NetApp Astra Control account:

  ./toolkit.py list apps -h
